# Arduino IOT Temperature Device

## Krav/opgaver

    - Modtage http req
        /SKRUOP
        /SKRUNED
        /gettemperature
    - Sende http req

    - Måler temperatur fra DS1621
    - Skrue op for varme (tænd rød led)
    - Skrue ned for varme (tænd blå red)

    - prioritere opgaver
        - måle i intervaller
        - lyttee efter http requests
